# Deepfake Audio Detector

## Overview
Deepfake Audio Detector is an AI-powered project that identifies whether an audio file is real or artificially generated (deepfake). The system uses machine learning and audio analysis techniques to detect manipulated speech and improve audio authenticity verification.

## Features
- Detects real and deepfake audio
- Audio feature extraction and analysis
- Machine learning based classification
- User-friendly interface
- Fast prediction results

## Technologies Used
- Python
- Flask
- Machine Learning
- Librosa
- NumPy
- Scikit-learn

## Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/Deepfake-Audio-Detector.git
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```
